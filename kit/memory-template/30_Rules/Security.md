# Security

## 保存しない情報

- APIキー、トークン、パスワード、秘密鍵
- 契約上保存できない顧客情報
- 公開範囲が不明な個人情報

## AIに読ませる前

- 利用するモデルとサービスのデータ取り扱いを確認する
- 公開情報、社内情報、機密情報を区分する
- 必要な範囲だけを作業フォルダへ置く

## 変更前

- 大量編集はテストVaultまたは小さな範囲で試す
- Git、Sync履歴、外部バックアップなど復元手段を用意する
- 削除と上書きは人へ確認する

## Community Plugins

- Restricted Modeから始める
- 目的、配布元、更新状況、権限を確認する
- 一度に複数導入せず、問題発生時に切り分けられるようにする
